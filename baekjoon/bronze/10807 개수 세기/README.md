**그동안 흔하게 본 유형이라 금방 논리를 세웠다**
> - 입력 받을 정수의 개수를 입력 받는다  
> - N 개의 정수를 배열에 저장한다  
> - 목표값 v를 받는다  
> - 배열을 순회하며 v를 만날 때 count(사진 속 i 는 실수! 정답을 저장할 변수를 하나 더 설정해야함)를 1증가시킨다.
